# 03 – Links and Images in HTML

This topic explains how to use **links** and **images** in HTML.
These elements are essential for navigation and visual content on web pages.

## File Included

- `03-links-and-images.html`  
  Demonstrates different types of links and image usage in HTML.

## HTML Links

Links are created using the `<a>` (anchor) tag.

### Types of Links Covered
- External links
- Links that open in a new tab
- Internal page links
- Section (bookmark) links
- Image as a link

### Common Attributes
- `href` – specifies the link destination
- `target="_blank"` – opens the link in a new tab

## HTML Images

Images are displayed using the `<img>` tag.

### Common Attributes
- `src` – path to the image file
- `alt` – alternative text if the image fails to load
- `width` / `height` – controls image size

## How to Use

1. Place an image file (e.g., `image.jpg`) in the same folder.
2. Open `03-links-and-images.html` in a browser.
3. Click links and observe image behavior.

## Purpose

This topic helps beginners learn:
- How navigation works in HTML
- How to add and manage images
- How links and images can work together
