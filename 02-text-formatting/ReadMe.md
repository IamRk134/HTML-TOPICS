## Text Formatting Elements Covered

The following types of text formatting are included:

- Bold and strong text
- Italic and emphasized text
- Underlined and highlighted text
- Small text
- Deleted and inserted text
- Subscript and superscript
- Preformatted text
- Blockquotes and inline quotations
- Abbreviations and citations
- Span element
- Line breaks and horizontal rules

## Purpose

The purpose of this file is to:
- Learn HTML text formatting tags
- Understand the difference between semantic and non-semantic tags
- Practice how text appears in a browser

## How to Use

1. Open `02-text-formatting.html` in any web browser.
2. Observe how each formatting tag changes the text.
3. Edit the text to experiment with different elements.
